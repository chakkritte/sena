Multi-Agent Orchestration

Automated plan → code → test → review → docs workflows. Coordinate specialized agents to solve entire development cycles autonomously.

Self-Evolution

Agents learn from past interactions and autonomously improve their strategies and rules, getting smarter with every session.

Claude Code-Style Chat

Modern terminal UX with multi-line input, draft preservation, persistent rendering, and interactive approval gates.

Browser Automation

Full web interaction and scraping via Playwright integration, enabling agents to research and debug across the web.

Observability First

Built-in OpenTelemetry tracing for every execution path. Monitor token costs, decision-making, and performance with granular detail.

Strict Engineering

100% Mypy compliance and strictly typed architecture designed for enterprise reliability and long-term maintainability.

Built for Scale

Agents (The Brains) Specialized logic for planning, coding, QA, and documentation.
Tools (The Hands) Secure adapters for Shell, Git, Filesystem, and Web Search.
Memory (The Recall) Persistent SQLite-backed long-term memory and context management.
Providers (The Connections) Agnostic abstraction layer for OpenAI, Anthropic, Gemini, and Ollama.
